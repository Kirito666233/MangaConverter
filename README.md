# MangaConverter
简易快速的将漫画转换为一本书
支持格式：JPG/PNG/WEBP
转换格式：PDF/EPUB

亮点：
有还看得过去的UI
支持单卷导入
文件夹批量拖拽导入
单卷导出/整本导出（同时输出总集）
ps:不建议输出总集，绝大多数情况会因为漫画太多输出后的pdf/epub会打开超级慢

本软件还在测试阶段，有问题请随时上传问题与截图至issue

已知问题：
在添加父文件夹时候若内部文件夹与内容过多可能会造成未响应情况，请优先使用拖拽进行操作
